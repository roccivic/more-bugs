# more-bugs

- puts bugs in your browser
- clicking on bugs kills them
- more bugs are spawned when one is killed
- click on 'no more bugs' to quit
- see the settings in ```lib/settings.js```
- launch demo: ```npm start```
- create build: ```npm dist```
